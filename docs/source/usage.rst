Usage
=====

Installation
------------

.. code-block:: console

   (.venv) $ git clone git@github.com:svmikurov/wselfedu.git
   (.venv) $ pip install poetry
   (.venv) $ poetry shell
   (.venv) $ poetry install
   (.venv) $ make migrate

Run app
-------

.. code-block:: console

   (.venv) $ make start
